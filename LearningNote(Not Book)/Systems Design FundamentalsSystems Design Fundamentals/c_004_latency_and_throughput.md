# latency and throughput

latency: 
- 資料在系統中運行的時間
- ex
  - clent to server
  - read from disk to memory
- 1MB, memory, 250ms
- 1MB, SSD, 1000ms
- 1MB, network, 10000ms
- 1MB, HDD, 20000ms
- a package , California -> Netherlands -> California, 150000ms

Throughput:
- 資料在系統中運行的速度(單位時間內處理的資料量->吞吐量)
- 增加坊吞吐量的方法
  - 多付點錢給雲服務商
    - 增加硬體
    - 分布式系統
